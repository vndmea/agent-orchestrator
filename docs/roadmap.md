# Roadmap

## Near term

1. Expand deterministic validation tooling and richer repository introspection.
2. Add more specialized worker capabilities and workflow branches.
3. Improve structured outputs for review and fix workflows.
4. Add CI workflows for typecheck, lint, and tests.

## Later

1. Domain-specific packages such as S1000D or XML toolchains.
2. More provider adapters and richer cost-aware routing.
3. Optional persistence and audit trails if product needs justify them.
