# Roadmap

## Near term

1. Add retention and inspection commands for worker execution records and artifact references.
2. Expand deterministic validation tooling and richer repository introspection.
3. Add more specialized task contracts through the shared registry.
4. Add CI workflows for typecheck, lint, tests, and smoke packaging.

## Later

1. Domain-specific packages such as S1000D or XML toolchains.
2. More provider adapters and richer cost-aware routing.
3. Stronger worker verification levels beyond interview and benchmark evidence.
